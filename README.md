# Our Financial Planner ✨

> **Aplikasi Keuangan Keluarga** — by Vina, Henry & Hiro

PWA (Progressive Web App) untuk mengelola keuangan rumah tangga. Bisa diinstall sebagai aplikasi di HP Android.

## 🏠 Fitur

- **Dashboard** — Ringkasan keuangan & monitoring pos uang
- **Transaksi** — Catat pemasukan & pengeluaran (sample data Juni 2026)
- **Budget Planning** — Anggaran per kategori
- **Laporan** — Laporan keuangan & over budget check
- **8 Pos Uang** — Blu, Comfort Life, dll
- **Panduan** — Cara input via chat

## 📦 Cara Install di HP Android

1. Buka URL PWA di Chrome Android
2. Tap **⋮** (menu) → **Install app / Add to Home screen**
3. Aplikasi akan terinstall seperti native app
